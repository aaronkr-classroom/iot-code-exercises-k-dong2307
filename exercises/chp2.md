# chp2 연습문제

1. 이 책에서는 마이크로파이썬 IDE로서 Thonny를 사용하고 있지만, 여러 가지 IDE가 존재합니다. 마이크로파이썬 IDE를 추가로 2개 이상 조사하고, Thonny와 장단점을 비교하세요.

   -> 첫번째 IDE는 PyCharm이다. PyCharm의 가장 큰 장점 중 하나는 스마트 코드 탐색이다. 이 기능을 사용하면 코드를 빠르고 효율적으로 탐색할 수 있다. 바로가기 키를 사용하여 코드의 특정 부분으로 이동하고 참조를 찾는 등의 작업을 수행할 수 있어 대규모 코드베이스로 작업하는 개발자들에게 큰 시간 절약 효과가 있다. 그리고 PyCharm은 오류 없는 코드 작성에 용이하다. 코드를 입력하는 동안 입력하는 내용과 일치하는 기능, 메서드 및 기타 코드 요소를 제안한다. 마지막으로 PyCharm은 통합 디버거를 사용하여 코드의 오류를 신속하게 식별하고 수정하는 데 도움을 준다.
   이러한 점들에서 PyCharm과 Thonny를 비교하자면 PyCharm은 코드 분석 기능이 매우 좋아 대형 프로젝트 관리에 적합하지만 무겁고 초보자가 쓰기에는 어렵다는 단점이 있다. 반대로 Thonny는 인터페이스가 단순하여 초보자가 쓰기에 좋고 가볍다는 장점과 대형 프로젝트에서는 쓰기 어렵다는 단점이 있다. 결국 PyCharm과 Thonny의 장단점은 완전히 반대라고 볼 수 있다.

  두번째 IDE는 VSCode이다. VSCode는 마이크로소프트에서 개발한 무료 오픈 소스 코드 편집기이다. VSCode의 장점으로는 어떤 기기에서든 사용할 수 있고, 저사양 하드웨어에서도 빠르게 실행되는 경량 코드 편집기이다. 또 VSCode의 인터페이스는 개인의 편의에 맞게 조절하여 개발자가 원하는 대로 코드 편집기를 조정할 수 있다. 이러한 이유 때문에 많은 사람들이 사용하고있다. 하지만 VSCode는 대규모 프로젝트를 위해 사용하기에는 어렵다는 단점이 있고 Python 코드를 작성하는 것은 가능하나, 코드를 실행하기 위해서는 컴퓨터에 Python 인터프리터가 설치되있어야 한다.
  Thonny와 VSCode를 비교하자면 두 IDE모두 가볍다는 공통점이 있으나, VSCode는 Thonny에 비해 초보자가 쓰기 어렵고, VSCode는 다양한 언어를 지원하고 실무에서 많이 사용된다는 차이점이 있다.

3. 마이크로파이썬의 펌웨어를 올리기 위해서 부팅할 때 별도의 버튼을 클릭하게 합니다. 해당 절차의 필요성을 설명하세요.

   -> 먼저 펌웨어는 하드웨어를 동작시키기 위해 내부에 저장된 기본 프로그램이다. 예를 들면 키보드에는 키 입력을 처리하는 프로그램이나 공유기에는 인터넷 연결을 제어하는 프로그램처럼 미리 들어가있는 프로그램을 얘기한다.
   본론으로 넘어가 마이크로파이썬의 펌웨어를 올리기 위해서 별도의 버튼을 클릭해야 하는 이유는 일반 실행 모드가 아니라 펌웨어 업로드 모드로 들어가기 위해서이다. 이것은 정상 실행과 업로드 모드를 구분하기 위함도 있고, 프로그램이 잘못 올라가서 실행이 안되는 경우 오류를 복구하기 위함도 있다. 쉽게말해 버튼은 실행 모드가 아니라 펌웨어 업로드 모드로 진입하기 위한 안전장치라 볼 수 있다.
